## 综述

## 安装

``` shell
tnpm install konph --save
```

## 网页加载

请运行一下命令生成打包文件：

``` bash
npm run build
```

生成的文件在.package目录中。

## 使用

请执行以下命令生成文档：

``` bash
npm run doc
```

生成的接口文档在doc目录中。
